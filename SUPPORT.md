## Common Questions

### How do I encrypt a file?
1. Open ByteDump
2. Click the "Encrypt" tab
3. Select a file using "Choose File..."
4. Set the number of encryption layers (1-10)
5. Enter a password for each layer
6. Click "Encrypt File"

### How do I decrypt a file?
1. Open ByteDump
2. Click the "Decrypt" tab
3. Select the encrypted file
4. Set the number of layers (must match encryption)
5. Enter the passwords in the same order as encryption
6. Click "Decrypt File"

### What encryption does ByteDump use?
ByteDump uses military-grade AES-256-GCM encryption with PBKDF2 key derivation.

### Is my data safe?
Yes! ByteDump:
- Encrypts files completely offline
- Never transmits data over the network
- Never stores your passwords
- Uses industry-standard encryption

### I forgot my password
Unfortunately, if you forget your encryption password, there is no way to recover your files. ByteDump uses strong encryption without any backdoors or password recovery mechanisms.
